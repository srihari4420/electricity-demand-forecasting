# Report

The written report for this project (introduction, data and preprocessing,
exploratory analysis, methods, evaluation design, results, error analysis,
discussion, limitations and conclusion) is kept here.

A full drafted version, with the figures and the model-comparison table filled
in from a pipeline run, is provided separately as `German_load_forecasting_report.docx`.
Export the figures from `outputs/figures/` into `reports/figures/` and reference
them from the report.
